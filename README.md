# HEARTH

HEARTH is a women-centered safety and mobility platform for Uzbekistan. It combines safer routing, trusted places, community reports, accessibility layers, emergency support, Umida help points, and the Hefi AI assistant.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production build

```bash
npm run build
npm start
```

Built with Next.js, React, and TypeScript.
